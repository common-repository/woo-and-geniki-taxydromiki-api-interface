Provides an interface between Woocommerce and Geniki Taxydromiki web service API. 

== Description ==

Provides interface with Geniki Taxydromiki web service API.
Description of this API can be found [here](http://voucher.taxydromiki.gr/help/jobservicesapi.pdf).

  
When a new order is completed, a remote  call is made to Geniki Taxydromiki to pick-up the package.
 
Also a voucher link is provided for printing.

== Frequently Asked Questions == 
= Does it work? =
Yes
= How =
You just use your credentials provided by Geniki Taxydromiki.
When a new order is completed, a remote  call is made to Geniki Taxydromiki to pick-up the package. Also a voucher link is provided for printing.
